### 專案簡介

模仿Steam上的bomb chicken，使用Java製作橫向2D闖關遊戲，並基於原本遊戲的玩法，額外設計雙人競賽模式，增添遊戲豐富度。

### 遊戲方式

透過推、疊、放炸彈來攻擊敵人並通過障礙物。遊戲中小雞需盡可能收集更多寶石來增加生命，也要努力安全抵達地圖終點。

### 操作方法

1P : ←、→左右移動、↓放炸彈

2P : A、D左右移動，S放炸彈
